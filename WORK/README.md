# Раздел 8: Я и планета (Экология и мир вокруг)

## Тема раздела

Раздел посвящён экологии и влиянию человека на окружающую среду.  
Цель — объяснить детям экологические проблемы и способы заботы о планете.

- сортировка мусора
- переработка отходов
- осознанное потребление

- изменение климата
